# Tutorial: Learning TADS 3

This is Eric Eve's *Beginner's Guide to TADS 3*, a hands-on tutorial that builds a complete interactive fiction game — *The Further Adventures of Heidi* — chapter by chapter. Each chapter introduces new features of the TADS 3 language and adv3 library through practical examples.

- **Chapter 1** — Introduction, installation, and programming basics
- **Chapter 2** — A very simple sample game
- **Chapter 3** — Rooms, objects, and vocabulary
- **Chapter 4** — Travel, puzzles, and controlling the action (verify, check, preconditions)
- **Chapter 5** — NPCs, conversation, and ending the game
- **Chapter 6** — Doors, windows, dark rooms, and hidden objects
- **Chapter 7** — Vehicles, shops, and cash transactions
- **Chapter 8** — Finishing touches, testing, debugging, and where to go next

Start with the [General Introduction](generalintroduction.md), or if you're already set up, skip ahead to [Chapter 2](chapter2.md).
