# TADS 3 Documentation

**TADS 3** (*Text Adventure Development System*) is one of the most powerful and sophisticated
interactive fiction authoring systems ever created. Originally developed by Michael J. Roberts,
TADS 3 provides a complete programming language, virtual machine, and adventure game library
for creating text-based interactive fiction.

## The Documentation

This site consolidates and modernizes the complete TADS 3 documentation, originally published
as separate HTML manuals between 2006 and 2016.
