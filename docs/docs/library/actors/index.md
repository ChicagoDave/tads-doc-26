# Actors

Actors are the characters that inhabit your game world — from the player character to shopkeepers, guards, and companions. The adv3 library provides a flexible framework for giving NPCs personality, dialogue, goals, and autonomous behavior.

- **[NPC Travel](npc-travel.md)** — Making NPCs move around the map on their own
